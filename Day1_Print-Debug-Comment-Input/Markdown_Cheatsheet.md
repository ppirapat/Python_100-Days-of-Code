# **Markdown Cheatsheet**

### **Headings**

- H1: #
- H2: ##
- H3: ###
- H4: ####
- H5: #####
- H6: ######
- Heading ID: ### (#custom-id)

### **Text**

- **bold** => emphasis
- _italic_ => slight emphasis
- **_bold italic_**
- ~~strikethrough~~
- <sub>subscript</sub> with tag: <sub>
- <sup>superscript</sup> with tag <sup>

### **Lists**

##### **Ordered List**

1. Step 1
2. Step 2

##### **Unordered List**

- Bullet (\*)
  - Bullet (\*)

* Bullet (-)
  - Bullet (-)

### **To-Do**

- [ ] To do
- [x] Done

### **Links and Images**

[Link Text](http://url)
![Alt Text](http://image_url)

### **Code**

`inline code`

### **Blockquotes**

> Note: This is a tip or important info
>
> > Sub-note or nested message

### **Horizontal lines**

- 3x -
- 3x \*
- 3x \_

### **LaTeX (Math)**

Inline: $E = mc^2$
Block:

$$
\int_a^b f(x)dx
$$

### **HTML**

<b>Bold using HTML</b>
<img src="url" width="200"/>

### **Table**

| Column 1 | Column 2 |
| -------- | -------- |
| Data A   | Data B   |
